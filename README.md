# CleanSweep

CleanSweep เป็นโปรเจ็คยูทิลิตี้สำหรับช่วยล้างไฟล์ขยะและไฟล์ชั่วคราวบนระบบ Windows (และรันจากซอร์สหรือไฟล์ที่คอมไพล์เป็น .exe)

**ความสามารถ (โดยย่อ)**

- ลบไฟล์ชั่วคราวและไฟล์ไม่จำเป็นเพื่อคืนพื้นที่ว่างบนดิสก์
- แสดงรายการไฟล์ที่จะลบก่อนยืนยันการลบ (dry-run/preview)
- รองรับการรันจากซอร์ส (`python main.py`) หรือจากไฟล์ที่คอมไพล์ในโฟลเดอร์ `dist/`

**การติดตั้ง**

1. ติดตั้ง Python 3.8+ (ถ้าต้องการรันจากซอร์ส)
2. ติดตั้ง dependencies:

```
pip install -r requirements.txt
```

**การใช้งาน (พื้นฐาน)**

- รันจากซอร์ส:

```
python main.py
```

- หากมีไฟล์ที่คอมไพล์ไว้ใน `dist/` (ตัวอย่าง `dist/CleanSweep.exe`) สามารถรันไฟล์ `.exe` โดยตรงบนเครื่อง Windows

**คำแนะนำการใช้งาน**

- ควรอ่านผลการสแกน/preview ก่อนยืนยันการลบ
- หากต้องการสำรองข้อมูล ให้เก็บไฟล์สำคัญก่อนใช้งานฟังก์ชันลบอัตโนมัติ
